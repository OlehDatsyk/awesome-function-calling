# Templates

Reusable starting points for defining new tools consistently across this repository's conventions.

- [`tool-definition-template.md`](./tool-definition-template.md) — blank tool spec following the 10-section format used throughout this repo.
- [`json-schema-template.json`](./json-schema-template.json) — a minimal, well-commented JSON Schema skeleton.
- [`error-handling-template.md`](./error-handling-template.md) — the standard structured-error shape used by every tool example.

Copy the relevant template when adding a new tool folder to keep documentation style consistent.
